# 🐾 Aibolit - Система управления ветеринарной клиникой

Полнофункциональная система для управления ветеринарной клиникой с поддержкой вакцинаций, приёмов, статистики и ML-диагностики.

---

## 🚀 Быстрый старт

```bash
# Запуск программы
python3 app.py
```

или просто дважды кликните на **`aibolit.bat`**

Откройте в браузере: **http://localhost:5000**

---

## ✨ Основные возможности

- 📋 **Управление клиентами** - карточки владельцев и животных
- 📅 **Расписание приёмов** - календарь с визуализацией
- 💉 **Учёт вакцинаций** - с автоматическими напоминаниями
- 💊 **База назначений** - лекарства и процедуры
- 📊 **Статистика** - графики и отчёты по всем параметрам
- 🤖 **ML диагностика** - автоматическая диагностика по симптомам
- 🌙 **Тёмная тема** - для комфортной работы
- 💾 **Автобэкап** - резервное копирование каждые 15 минут

## 📦 Установка

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/Stuard256/Aibolit.git
cd Aibolit
```

### 2. Установите зависимости
```bash
python3 -m pip install -r requirements.txt
```

### 3. Запустите программу
```bash
python3 app.py
```

Подробная инструкция: **[docs/УСТАНОВКА.md](docs/УСТАНОВКА.md)**

## 🎨 Функции

### Управление данными
- ✅ Карточки владельцев с контактной информацией
- ✅ Карточки животных с полной историей
- ✅ Поиск по всем параметрам
- ✅ Импорт/экспорт данных из CSV

### Приёмы и вакцинации
- ✅ Календарь приёмов с визуализацией
- ✅ Учёт вакцинаций по типам
- ✅ Автоматические напоминания о прививках
- ✅ Генерация PDF отчётов о вакцинации

### Статистика и аналитика
- ✅ Графики по вакцинациям
- ✅ Анализ по видам животных
- ✅ Финансовые отчёты
- ✅ Выбор периода для анализа
- ✅ Статистика по месяцам

### Дополнительно
- ✅ ML диагностика заболеваний
- ✅ Калькулятор стоимости лечения
- ✅ База лекарств и процедур
- ✅ Переключение тёмной/светлой темы
- ✅ Адаптивный дизайн для мобильных

---

## 🛠️ Технологии

- **Backend:** Flask, SQLAlchemy, APScheduler
- **Frontend:** Bootstrap 5, jQuery, Chart.js
- **ML:** scikit-learn, numpy, pandas
- **Database:** SQLite
- **PDF:** wkhtmltopdf, pdfkit
- **Documents:** python-docx, docxtpl
